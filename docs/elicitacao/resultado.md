## Introdução
Nem todos os requisitos devem ser imediatamente implementados. Por isso existe
um processo de priorização de requisitos, no qual são definidos aqueles que
têm mais importância no momento e, portanto, devem ser implementados primeiro.

A técnica de priorização utilizada foi a de R$ 100. Consiste em imaginar que o
orçamento do projeto é de R$ 100 para realizar todos os requisitos. Então cada 
requisito é analisado e discutido qual o seu preço, quanto reais a equipe está
disposta a pagar para "comprá-lo". Quanto maior essa quantidade, maior é a prioridade
do requisito.

## Metodologia
Em reunião no Discord, os requisitos levantados pelos participantes foram colocados em uma planilha do
Google Sheets, um em cada linha. Em seguida, a cada requisito foi atribuído uma
quantia em reais.
