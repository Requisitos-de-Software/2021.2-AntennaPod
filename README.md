# AntennaPod

[Site de documentação](https://requisitos-de-software.github.io/2021.2-AntennaPod/).

## Configuração do ambiente

Dependências:

- Python 3
- Pip 3
- [MkDocs](https://www.mkdocs.org/)

```py
pip3 install -r requirements.txt # ou com pip
```

Subir o servidor:

```py
mkdocs serve
```
